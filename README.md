## 资源详情

- **正面情绪词**：包含了表达积极、愉快、满意等正向情感的词语，有助于识别文本中的乐观态度。

- **程度副词**：如“非常”、“稍微”，用以增强或减弱情感的强度，对于细腻的情感分析不可或缺。
